Search box enchancements service.

== Description ==

_Search+_ enchances your serch box, making it more convenient and easy to use.

Major features in _Search+_ include:

* Voice recognition
* Type ahead
* Did you mean?
* Transliteration
* Usage statistics
* Voice navigator (experimental)
* Jump links (experimental)

Once the plugin is activated, you will have to go to the _Search+_ configuration page (from the left menu) and complete the account creation - it takes a single click!

A step by step guide is available at [our wiki](https://searchplus.pro/wiki/en-US/index.php?title=WordPress).


== Installation ==

Obtain the SearchPlus plugin, activate it and then register or link your existing registration to the current site with a single click.

Extensive configuration options are available at your account's settings section
in the [customer portal of the service](https://searchplus.pro/customer).

More information is available in [our wiki](https://searchplus.pro/wiki/en-US).

== Frequently Asked Questions == 

= I just want to get the microphone for free. Is there a way? =

Sure, by default the service uses a free plan, which gives you the most popular feature - the microphone (voice recognition).

= What are the experimental features? =

They are generally available, but the access to them is restricted since we are still running usability and user acceptance tests.
